CreateSimulator README:

The CreateSimulator is intended to provide a simple editing and testing environment for iRobot's Create educational platform, in particular for the Command Module that can plug into the back.

It will provide three methods for testing code:

  1. Direct compilation and downloading to the Command Module for running code on the embedded system.
  2. Indirectly running code on the Create (or any serial-equipped iRobot device compatible with their Open Interface v2) via a serial cable.
  3. Simulated execution on a host computer in a virtual testing environment.
